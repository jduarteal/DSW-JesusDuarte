El objetivo del sitio web es introducir a un joven, adulto y/o una persona del común en el mundo del HTML, explicandole de una manera practica y con terminología el desarrollo de una pagina web.

De igual manera se definirán los terminos y estructuras requeridas para realizar de un desarrollo web, validando fuentes e imagenes siendo atractiva para el usuario final.
